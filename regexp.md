# regexp

## negative/positive look-ahead/look-behind

- link: `https://stackoverflow.com/questions/2973436/regex-lookahead-lookbehind-and-atomic-groups`
- table
  ```txt
  (?!) - negative lookahead
  (?=) - positive lookahead
  (?<=) - positive lookbehind
  (?<!) - negative lookbehind

  (?>) - atomic group
  ```
